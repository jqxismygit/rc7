# 底部导航栏图标获取指南

## 问题说明

微信小程序的 tabBar 需要真实的 PNG 图片文件，目前项目中的图标是空文件，需要替换为真实图标。

## 快速解决方案

### 方案一：使用 iconfont（推荐）

1. 访问阿里巴巴矢量图标库：https://www.iconfont.cn/

2. 搜索并下载以下图标：
   - 搜索"首页" 或 "home" - 下载一个房子图标
   - 搜索"票券" 或 "ticket" - 下载一个票券图标  
   - 搜索"我的" 或 "user" - 下载一个用户图标

3. 下载要求：
   - 格式：PNG
   - 尺寸：81px × 81px（推荐）
   - 颜色：灰色（未选中）、绿色或蓝色（选中）

4. 重命名并替换：
   ```
   下载的首页图标 → tab-home.png（灰色）
   下载的首页图标 → tab-home-active.png（彩色）
   下载的票券图标 → tab-ticket.png（灰色）
   下载的票券图标 → tab-ticket-active.png（彩色）
   下载的用户图标 → tab-profile.png（灰色）
   下载的用户图标 → tab-profile-active.png（彩色）
   ```

5. 将这6个文件放入 `static/` 目录

### 方案二：使用在线工具生成

访问：https://www.iconfinder.com/ 或 https://icons8.com/
- 搜索对应图标
- 下载 PNG 格式
- 调整尺寸为 81x81
- 重命名后放入 static 目录

### 方案三：使用 Emoji 表情（临时方案）

如果暂时没有图标资源，可以使用 emoji 表情：

1. 访问：https://emojipng.com/
2. 下载以下 emoji：
   - 🏠 (house) → tab-home.png
   - 🎫 (ticket) → tab-ticket.png  
   - 👤 (user) → tab-profile.png
3. 使用图片编辑工具调整颜色制作选中状态

### 方案四：暂时不使用 tabBar

如果暂时不需要底部导航栏，可以在 pages.json 中注释掉 tabBar 配置（已完成）。

页面间跳转可以使用：
```javascript
// 跳转到首页
uni.navigateTo({ url: '/pages/index/index' })

// 跳转到票夹
uni.navigateTo({ url: '/pages/my-tickets/my-tickets' })

// 跳转到个人中心
uni.navigateTo({ url: '/pages/profile/profile' })
```

## 恢复 tabBar 配置

当你准备好图标文件后，在 pages.json 中添加以下配置（在 pages 数组后面）：

```json
"tabBar": {
  "color": "#7A7E83",
  "selectedColor": "#667eea",
  "borderStyle": "black",
  "backgroundColor": "#ffffff",
  "list": [
    {
      "pagePath": "pages/index/index",
      "iconPath": "static/tab-home.png",
      "selectedIconPath": "static/tab-home-active.png",
      "text": "首页"
    },
    {
      "pagePath": "pages/my-tickets/my-tickets",
      "iconPath": "static/tab-ticket.png",
      "selectedIconPath": "static/tab-ticket-active.png",
      "text": "票夹"
    },
    {
      "pagePath": "pages/profile/profile",
      "iconPath": "static/tab-profile.png",
      "selectedIconPath": "static/tab-profile-active.png",
      "text": "我的"
    }
  ]
}
```

## 图标设计建议

- **尺寸**：81px × 81px（推荐）或 40px × 40px
- **格式**：PNG（支持透明背景）
- **颜色**：
  - 未选中：#7A7E83（灰色）
  - 选中：#667eea（紫蓝色）或 #3cc51f（绿色）
- **风格**：简洁、扁平化、线性图标

## 注意事项

1. 图标文件必须是真实的 PNG 图片，不能是空文件
2. 图标路径相对于项目根目录
3. tabBar 最少需要2个，最多5个标签
4. 图标大小建议不超过 40kb
5. 建议使用透明背景的 PNG 图片
