# 静态资源说明

## 图标资源

由于这是演示项目，底部导航栏的图标需要手动添加。请准备以下图标文件：

### 底部导航栏图标（建议尺寸：81px × 81px）

1. **tab-home.png** - 首页图标（未选中）
2. **tab-home-active.png** - 首页图标（选中）
3. **tab-ticket.png** - 票夹图标（未选中）
4. **tab-ticket-active.png** - 票夹图标（选中）
5. **tab-profile.png** - 我的图标（未选中）
6. **tab-profile-active.png** - 我的图标（选中）

### Logo

- **logo.png** - 应用 Logo（建议尺寸：200px × 200px）

## 临时解决方案

如果暂时没有图标资源，可以：

1. 使用纯色占位图
2. 使用 emoji 表情
3. 使用 iconfont 图标库
4. 从设计资源网站下载免费图标

## 推荐图标资源网站

- iconfont (阿里巴巴矢量图标库): https://www.iconfont.cn/
- IconPark (字节跳动图标库): https://iconpark.oceanengine.com/
- Flaticon: https://www.flaticon.com/
- Icons8: https://icons8.com/

## 使用说明

将准备好的图标文件放入 `static/` 目录即可。项目会自动引用这些图标。
