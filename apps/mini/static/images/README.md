# 静态图片资源说明

本目录存放设计稿中的静态图片，格式为 **PNG** 或 **JPG**。

## ⚠️ 微信主包 2MB 限制（必读）

微信小程序**主包体积不能超过 2MB**。以下三张图在首页使用，会打进主包，**单张建议控制在 500KB 以内**（三张合计约 1.5MB 以内，留出代码等空间）：

| 文件 | 当前易超限 | 建议 |
|------|------------|------|
| `hero-banner.png` | 约 1.3MB 会超限 | 压缩后 <500KB，宽度建议 ≤750px |
| `hero-banner-2.png` | 约 1.3MB 会超限 | 压缩后 <500KB，宽度建议 ≤750px |
| `event-card.png` | 约 1.3MB 会超限 | 压缩后 <500KB，宽度建议 ≤680px |

**压缩方式（任选其一）：**

1. **TinyPNG**：https://tinypng.com/ 上传 PNG 压缩后替换原文件。
2. **Figma 导出**：导出时选 **JPG** 或降低 PNG 分辨率（如 1x），或导出宽度 750px 再压缩。
3. **本地工具**：用 Squoosh、ImageOptim 等做有损/无损压缩后替换。

替换并保存后重新编译小程序，主包体积即可降到 2MB 以内。

## 如何从 Figma 导出

1. 在 Figma 设计稿中选中对应图层/帧。
2. 右侧面板选择 **Export**，格式选 **PNG** 或 **JPG**。
3. 勾选 **2x** 或 **3x** 以适配高清屏（可选）。
4. 导出后重命名为上表文件名，覆盖本目录下同名文件即可。

当前目录下为 1×1 像素占位图，仅用于占位；替换为正式切图后页面将显示正确图片。
