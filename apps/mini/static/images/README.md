## 如何从 Figma 导出

1. 在 Figma 设计稿中选中对应图层/帧。
2. 右侧面板选择 **Export**，格式选 **PNG** 或 **JPG**。
3. 勾选 **2x** 或 **3x** 以适配高清屏（可选）。
4. 导出后重命名为上表文件名，覆盖本目录下同名文件即可。

当前目录下为 1×1 像素占位图，仅用于占位；替换为正式切图后页面将显示正确图片。
